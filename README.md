UX Rocket Modal
==============
A multi window capable modal plugin. It can easily customizable via its settings and styles.

Sample usage

```html
<a href="#hidden-content" class="modal">Open Modal</a>
```

```js
// with defaults
$('.modal').modal();

// or with your settings
$('.modal').modal({
    position: fixed,
    maxWidth: '640px',
    maxHeight: '90%'
});
```
